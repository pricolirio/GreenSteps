## Dependencias y Servicios de Terceros

Este proyecto utiliza los siguientes servicios de terceros:

1. **Firebase Authentication**: Proporciona autenticación de usuario. Para más detalles, consulta los [Términos de Servicio de Firebase](https://firebase.google.com/terms).
2. **Firebase Firestore**: Proporciona una base de datos en la nube. Para más detalles, consulta los [Términos de Servicio de Firebase](https://firebase.google.com/terms).
3. **Google Maps Platform API**: Proporciona servicios de mapas y localización. Para más detalles, consulta los [Términos de Servicio de Google Maps Platform](https://cloud.google.com/maps-platform/terms).
